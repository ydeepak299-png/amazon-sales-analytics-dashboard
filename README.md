# 📊 Amazon Sales Analytics Dashboard

An end-to-end **Business Intelligence** project that analyzes Amazon sales data using **Excel, SQL, Power Query, Power BI, and DAX**. The project transforms raw sales records into an interactive dashboard that helps stakeholders monitor business performance, identify sales trends, and make data-driven decisions.

---

# 🚀 Project Overview

This project demonstrates the complete Data Analytics workflow:

- Data Cleaning using Excel & Power Query
- Data Analysis using SQL
- Data Modeling in Power BI
- DAX Calculations
- Interactive Dashboard Design
- Executive Summary Reporting
- Drill-through Analysis
- Report Page Tooltips
- Business Insight Generation

The dashboard is designed for business users who need quick insights into sales performance, top-performing categories, regional sales, monthly trends, and order fulfillment.

---

# 🎯 Business Objectives

The dashboard helps answer important business questions such as:

- Which product category generates the highest revenue?
- Which state contributes the highest sales?
- What is the monthly revenue trend?
- What is the overall cancellation rate?
- Which courier status occurs most frequently?
- Which month generated the highest revenue?
- How can executives quickly monitor overall business performance?

# 📂 Dataset

The project is based on approximately **129,000 Amazon sales records**.

The dataset contains information such as:

- Order Date
- Product Category
- Ship State
- Courier Status
- Sales Channel
- Quantity
- Revenue Amount
- Order Status

> **Note:**  
> The cleaned dataset is **not included** in this repository because it exceeds GitHub's standard file size limit (100 MB).

# 🧹 Data Cleaning

The dataset was cleaned before analysis using Excel and Power Query.

Cleaning steps included:

- Standardized column names
- Removed inconsistencies
- Checked duplicate records
- Standardized date format
- Handled missing values
- Cleaned categorical fields
- Created Month Name
- Created Month Number
- Prepared data model for Power BI

# 📌 Business Insights

Some key findings from the analysis:

- **Set** is the highest revenue-generating product category.
- **Maharashtra** contributes the highest revenue among all states.
- **April** recorded the highest monthly revenue.
- **Shipped** is the dominant courier status.
- Overall cancellation rate is **14.21%**, highlighting opportunities to reduce cancelled orders and improve fulfillment.
